# Contest Plan

## Objective

Build and submit a polished utility-first watchface for the Spring 2026 Pebble App Contest.

## Positioning

- Primary judging goal: feel complete, legible, and useful
- Primary platform goal: first-class support for `emery` and `gabbro`
- Product goal: make a strong first impression in screenshots and daily use

## Release Strategy

1. Build a stable MVP with time, date, battery, and Bluetooth status
2. Add one premium differentiator after the base is solid
3. Capture screenshots for each supported platform
4. Publish publicly in the Pebble appstore during the contest window

## Quality Bar

- Fast visual scan for the current time
- Clean layout on round and rectangular screens
- No placeholder copy
- Repo ready for judging and source review
